01

Your first five minutes

Use the studio as a tight create–observe–debug loop.

  1. Choose a lessonStart with the butterfly, kinetic fractal, or flight study.
  2. Change one valueTry a hue, recursion depth, speed, or wing amplitude.
  3. Run and observeAuto-run waits briefly while you type; Run executes immediately.
  4. Read the consoleLogs, syntax errors, runtime errors, and rejected promises appear below the canvas.
