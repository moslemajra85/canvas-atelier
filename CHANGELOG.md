## 1.0.0 — 2026-07-16

- Added Worker and OffscreenCanvas execution with Stop and a five-second initialization watchdog.
- Added a reusable fractal, particle, texture, and user-image library.
- Added IndexedDB asset storage, SVG sanitization, license notes, and portable project packaging.
- Added portable composition layers with visibility, opacity, blend, order, and removal controls.
- Added exact-size PNG, JPEG, and WebP export plus browser-recorded WebM animation export.
- Added runtime autocomplete, semantic diagnostics, CSP restrictions, and keyboard navigation hardening.
- Upgraded the project format to version 2 while retaining version 1 import compatibility.
